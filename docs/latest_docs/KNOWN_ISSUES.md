## 第 1 部分：已知问题

### 1.1 数据准确性

#### ⚠️ 问题: EUB Excel 更新延迟

**影响程度**: 🟡 中
**状态**: 已确认
**描述**:
- NBEUB 官网 Excel 通常在周四下午 13:00 Atlantic 发布
- 但有时延迟 1-2 小时
- 如果 GitHub Actions 在发布前运行，会抓取旧数据

**当前解决方案**:
- GitHub Actions 运行两次 (08:00 UTC 和 22:30 UTC)
- 22:30 UTC 通常能捕获延迟的数据

**长期方案**:
- [ ] 改为订阅官方 API (如果开放)
- [ ] 增加重试机制或队列

---

#### ⚠️ 问题: 汇率数据偶尔缺失

**影响程度**: 🟡 中
**状态**: 已确认
**描述**:
- Yahoo Finance 在某些节假日不提供 CAD 汇率
- 系统使用 `ffill()` 用前一天汇率填充

**当前解决方案**:
```python
market_df['cad_rate'] = market_df['cad_rate'].ffill()
```

**风险**:
- 长假期间汇率可能不准确
- 需要人工检查

**长期方案**:
- [ ] 集成多个汇率数据源
- [ ] 本地缓存汇率
