# NB Gas Price Dashboard - 下一阶段优化路线图

本报告由系统架构师、资深程序员、UI/UX 专家、测试工程师及产品经理联合评审，旨在为项目的未来发展提供专业建议。

---

## 1. 产品经理 (PM) 视角：商业价值与用户增长
**核心目标：从“工具”进化为“生活必需品”**

*   **P0 加油建议指数 (Refuel Meter)**：
    *   增加一个视觉化的“红绿灯”指示。不仅仅显示涨跌，更要直接告诉用户：“🔴 今晚加满”或“🟢 等到周四”。
*   **P1 多能源品种支持 (Multi-Fuel)**：
    *   扩展监控范围至 **Diesel (柴油)** 和 **Heating Oil (取暖油)**。这对 NB 省的大量卡车司机和家庭住户具有巨大的吸引力。
*   **P2 社交分享图片 (Social Sharing)**：
    *   增加“生成周报图片”功能。用户可一键生成精美的预测长图，发到社区群组，利用社交裂变获取用户。
*   **P2 大西洋省份对比 (Regional Radar)**：
    *   引入 Nova Scotia 和 PEI 的价格，实现省际价格对比，吸引跨省通勤者。

---

## 2. 系统架构师 (System Architect) 视角：可扩展性与高可用
**核心目标：支持大规模并发与多省份扩展**

*   **数据隔离与 API 化**：
    *   目前的 `data.json` 耦合了所有数据。建议将历史数据与当日实时数据分离，或按省份拆分 JSON，以提升首屏加载性能。
*   **多源数据冗余 (Data Redundancy)**：
    *   目前完全依赖 Yahoo Finance。建议引入第二数据源（如 Investing.com 或专业的 Oil Price API）作为 Fallback，防止单一 API 挂掉导致预测失效。
*   **CDN 深度优化**：
    *   在 Cloudflare 中配置更激进的 `Cache-Control` 策略，确保在 GitHub Actions 运行间隙，全球用户都能从边缘节点瞬间加载数据。

---

## 3. 资深程序员 (Senior Developer) 视角：代码健壮性与工程规范
**核心目标：实现零人工干预的自动化系统**

*   **专业日志系统 (Professional Logging)**：
    *   将 `print` 语句替换为 Python 的 `logging` 库。记录详细的抓取耗时、API 状态码以及预测算法的中间变量，便于复盘误差。
*   **类型安全 (Type Hinting)**：
    *   为 Python 脚本增加 Type Hints，并引入类型检查工具，防止在处理复杂的金融 DataFrame 时出现隐式错误。
*   **配置文件增强**：
    *   将 `settings.py` 升级为支持多环境（Dev/Prod）的配置，或者使用环境变量（Secret）管理敏感的监控配置。

---

## 4. UI/UX 专家视角：精细化交互与品牌感
**核心目标：打造极致的移动端“App 感”**

*   **骨架屏加载 (Skeleton Loader)**：
    *   在 `data.json` 加载完成前，展示与卡片/图表轮廓一致的灰色动画占位符，消除文字闪烁感。
*   **图表交互降噪**：
    *   默认隐藏“Retail Spread”曲线，仅在用户点击图例时激活。减少初屏视觉负担。
*   **PWA 离线能力**：
    *   优化 `sw.js`，实现真正的离线缓存。即使在加油站等网络信号极差的地方，用户也能打开网页查看最后一次同步的价格数据。

---

## 5. 测试工程师 (QA) 视角：数据准确性与回归验证
**核心目标：确保预测算法的公信力**

*   **算法回归测试 (Backtesting Suite)**：
    *   编写测试脚本，用过去 12 个月的历史数据跑一遍预测模型，计算预测值与实际值的“命中率”和“平均误差”，并持续优化参数。
*   **CI 自动化测试**：
    *   在 GitHub Actions 中集成 `pytest`。在每次推送代码时，自动验证预测逻辑和 Excel 解析函数是否被破坏。
*   **模拟数据测试 (Mocking)**：
    *   模拟极端市场情况（如国际油价腰斩或暴涨），验证 UI 在极端数值下的展示（如文字溢出、卡片崩溃等问题）。

---

## 🛠️ 建议优先执行顺序 (Priority List)

1.  **P0**: 加油建议指数 (直接提升用户决策价值)
2.  **P0**: 算法回归测试 (确保预测准确性，这是产品的根基)
3.  **P1**: 骨架屏与 UI 降噪 (提升第一眼好感度)
4.  **P1**: 引入柴油监控 (扩大受众群体)
